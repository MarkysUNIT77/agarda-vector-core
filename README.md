# agarda-vector-core
Суверенное ядро нелинейного инференса вселенной A.G.A.R.D.A. Легковесный векторный поиск на чистом Python/NumPy с экстремальной RAM-оптимизацией через memmap. Очищено от корпоративного шума, моков и тяжелых CUDA-зависимостей. Скорость: >6 млн сканирований в секунду на голом железе.
