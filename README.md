# agarda-vector-core

Суверенное ядро нелинейного инференса вселенной A.G.A.R.D.A. Легковесный векторный поиск на чистом Python/NumPy с экстремальной RAM-оптимизацией через memmap. Очищено от корпоративного шума, моков и тяжелых CUDA-зависимостей.

## Физические показатели (Честный бенчмарк на железе)

## Как запустить локально

```bash
git clone https://github.com
cd agarda-vector-core
pip install -r requirements.txt
python vector_core.py
```
